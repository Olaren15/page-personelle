# Page accueil

Le code source de ma page personnelle (qui est disponible au https://cathgilbert.com)

## Développement local

### Outils nécessaires :

- Node JS
- Npm

1. Installer les dépendances

   ```bash
   npm install
   ```

2. Démarrer le serveur de développement

   ```bash
   npm run dev
   ```

# Technologies utilisées

- Astro
- Typescript
- Tailwind CSS
